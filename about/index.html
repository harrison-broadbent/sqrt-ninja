About

Sqrt Ninja is a fun toy project I built, mainly because it didn't already exist.

It uses the big.js javascript library for the square root approximation.

The limit of 1000 decimal places really is arbitrary.

I chose it so that your device won't lag too much, but there's no reason why you can't go bigger.

Eventually though, it would take prohibitively long to complete the calculations.
